

[344. 反转字符串 - 力扣（LeetCode）](https://leetcode.cn/problems/reverse-string/description/)





```java
    class Solution {
        public void reverseString(char[] s) {
            for (int i = 0, j = s.length - 1; i < s.length / 2; i++, j--) {
                swap(s, i, j);
            }
        }

        public void swap(char[] s, int i, int j) {
            char temp = s[i];
            s[i] = s[j];
            s[j] = temp;
        }
    }
```
